CSS
===

Pegasus lets you choose from three different CSS frameworks:
`Tailwind CSS <https://tailwindcss.com/>`_, `Bootstrap 5 <https://getbootstrap.com/>`_,
and `Bulma <https://bulma.io/>`_.
Every Pegasus theme works on desktop and mobile can can be customized to match your specific site/brand.

Read on for help on choosing a CSS theme, and documentation on working with CSS in Pegasus.
